# [ShellShockLite]

A game made in **Pico-8** (Fantasy Console).
The game was programmed as a part of a project for the university course Game Design & Development and isn't finished yet. 
The game is a 2D platformer with shooting mechanics and destructible terrain.
---

## 🎮 **How to Play**

### Controls
- **↑↓← → (Arrow Keys)**: Move
- **X**: Shoot
- **C**: Change weapon (Normal Shot / Grenade Shot)

## ⚙️ **Features**

## 🛠️ **Configuration**
Edit the config section to adjust:
- Player speed (`PLAYER_SPEED`)
- Jump height (`JUMP_HEIGHT`)
- Weapon settings (damage, cooldown, etc.)
- Gravity (`GRAVITY`)

```lua
